Gravity Flow Planter: Gravity-Fed Modular Garden
A 6ft x 8ft modular planting system designed for gravity-based irrigation.
 Overview
This project is a large-scale, ground-supported planter designed to maximize vertical space for low-profile crops like carrots, cabbages, and leafy greens.
The core feature is a cascading water system:
Top-Down Irrigation: Water added to the highest tier drips through precision-modeled drainage holes.
Self-Sustaining: Lower levels receive moisture from the tiers above, ensuring even distribution through the soil and reducing water waste.
Technical Specifications
Dimensions: 6ft (H) x 8ft (W)
Support: Ground-standing integrated leg system.
Soil Depth: Optimized for root vegetables (carrots) and head-forming vegetables (cabbage).
Software: SolidWorks
Manufacturing Options
This design is versatile and can be produced in two primary ways:
1. 3D Printing (Modular)
Material: PETG or ASA (UV resistant for outdoor use).
Strategy: Due to the 8ft width, the model should be exported as individual segments and joined using exterior-grade adhesive or stainless steel bolts.
Infill: 25% Gyroid for maximum structural strength against soil weight.
2. Wood Construction
Material: Cedar or Redwood (natural rot resistance).
Assembly: Use the .STEP files to pull dimensions for timber framing.
 File Structure
